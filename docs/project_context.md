# Project Context

## Why this repository exists

This project began as an AP Research paper on perceived stress differences between AP STEM and AP Humanities students. The original study received a score of 5 and used survey-based data collection with inferential statistics.

The repository now reframes that work as **Academic Stress Analytics** — a data analytics portfolio project. The AP Research study is **Case Study #1**, not the entire scope of the project.

## Portfolio goals

1. Demonstrate end-to-end data analysis skills: cleaned data, Python notebooks, SQL, and visualizations.
2. Be transparent about data limitations (cleaned export only; raw surveys unavailable).
3. Avoid overclaiming generalizability beyond the studied school and this sample.
4. Provide a foundation for future case studies using external student wellness datasets.

## Roadmap (future work)

- Integrate external datasets on student stress to compare this sample with broader populations.
- Build dashboards (e.g., Power BI) for exploratory stress trends.
- Add predictive modeling only when appropriate data and validation design are available.

Any extension should clearly separate **Case Study #1** (single-school AP sample) from analyses based on external data.

## Academic integrity note

This repository does **not** claim full reproducibility from raw survey responses. It documents and re-analyzes a cleaned, anonymized dataset generated from the original study.
