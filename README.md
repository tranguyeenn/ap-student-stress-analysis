## Academic Context

This project is based on an AP Research study that received a score of 5.  
The original paper investigated differences in perceived stress levels between AP STEM and AP Humanities students using a modified Perceived Stress Scale (PSS).

The research included:
- Survey-based data collection
- Statistical analysis using a 2-sample t-test
- Interpretation grounded in psychological theories of stress (e.g., Yerkes-Dodson Law, cognitive-motivational model)

This repository focuses on transforming the original research dataset into a structured, analysis-ready format and extending it into a data analysis project.

## Future Work

- Integrate external datasets on student stress to compare AP students with broader populations
- Build an interactive Power BI dashboard to visualize stress trends and distributions
- Expand analysis to include additional variables (e.g., workload, academic background)
- Reproduce and validate statistical findings using Python or SQL-based workflows
